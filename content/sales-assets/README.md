# Sales Assets — Vanguardeer
*Internal use only. Not linked publicly from the website.*

## Files in This Directory

| File | Purpose | When to Use |
|------|---------|-------------|
| `revenue-leak-brief-template.md` | Template for the Revenue Leak Intelligence Brief diagnostic | After qualifying a prospect via the brief form |
| `clinic-revenue-leak-brief-template.md` | Clinic-specific version of the Revenue Leak Brief | For private clinic prospects |
| `cyber-gtm-diagnostic-template.md` | GTM diagnostic for cybersecurity/MSP firms | For cyber/IT firm prospects |
| `market-capture-sprint-outline.md` | Full programme outline for the 12-week sprint | For proposals and client briefings |
| `sales-call-prep-template.md` | Pre-call intelligence brief | Before every discovery or sales call |
| `objection-handling-guide.md` | Responses to common objections | During sales conversations |
| `outbound-message-sequences.md` | Outreach email/LinkedIn templates | For proactive outbound prospecting |

## Usage Notes

1. Never share these files directly with prospects.
2. Use the brief templates to build branded PDF/DOCX deliverables.
3. Personalise all outbound templates — do not send generic versions.
4. Update templates when messaging or offer structure changes.
5. All templates assume premium positioning. Do not dilute with cheap language.
